# 👥 HR Employee Attrition Analysis
### Google Sheets Project | Eleazar Soto

## 📋 Project Overview
Analysis of employee attrition using the IBM HR Analytics dataset.
Identifying key factors that contribute to employee turnover.

## 📊 Dataset
- **Source:** IBM HR Analytics
- **Records:** 1,470 employees
- **Variables:** 35 columns including Age, Department, Job Role, Salary, Attrition

## 🔍 Key Questions Analyzed
- What is the overall attrition rate?
- Which departments have the highest turnover?
- Does salary influence attrition?
- What is the relationship between job satisfaction and attrition?
- Which age groups are more likely to leave?

## 🛠️ Tools & Techniques
- Google Sheets
- Pivot Tables
- Charts and Visualizations
- Data Cleaning

## 📁 File Structure
- `HR-Employee-Attrition.xlsx` — Main analysis file
  - Sheet 1: Raw data
  - Sheet 2: Analysis
  - Sheet 3: Pivot Tables
  - Sheet 4: Charts

## 💡 Key Findings
- Overall attrition rate: **16.1%** (237 out of 1,470 employees)
- **Sales** department has the highest turnover rate at 20.6%
- Employees who leave are **4 years younger** on average (33.6 vs 37.6)
- Employees who leave earn **30% less** ($4,787 vs $6,832/month)
- **53.6%** of employees working overtime tend to leave
- Lower job satisfaction strongly correlates with attrition
